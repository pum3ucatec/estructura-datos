### Crear un commit

1. Estado del repositorio

```
git status
```
2. Agregar las carpetas y/o archivos
```
git add .
```
3. Crear el commit
```
git commit -m "Mi primero commit"
```
4. Subir el commit al repositorio remoto
```
git push
```
